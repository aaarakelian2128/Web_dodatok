# Web_dodatok
# Лабораторна робота №1
## Тема: Розробка статичного інтерфейсу Web-додатка 

### Інформація про студента
* **ПІБ:** Аракельян Альбіна Андріївна
* **Група:** КВ-32

### Завдання до роботи 
Розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру.

**TaskFlow** — це веб-додаток для персонального тайм-менеджменту, розроблений з акцентом на мобільний інтерфейс (Mobile First). Проєкт дозволяє користувачам реєструватися, керувати своїми справами у зручному воркспейсі та налаштовувати особистий профіль.

**Функціональні вимоги до інтерфейсу:**.
* Реєстрація користувача (поля: ім'я, email, стать, дата народження).
* Вхід до сайту (поля: email, пароль).
* Профіль користувача (дані у табличному вигляді).
* Головна сторінка (емблема та опис).
* Робоча сторінка додатка: панель додавання та перегляду завдань зі статусами.
